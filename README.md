# Neon Rush (3D Endless Runner Prototype)

A fast-paced 3D endless runner where speed and reflexes are the key to survival in a futuristic city. This project was developed as the Final Project for the "Video Game Creation" course.

## üìñ About The Project
**Neon Rush** is a functional prototype that puts the player in control of an agile robot running through a procedurally generated track. The objective is simple: dodge the obstacles that appear on the path by jumping and moving sideways to survive as long as possible and set a new high score.

This project was conceived and developed over one semester as the final project for the Video Game Creation course at UTEZ, under the guidance of Professor Lic. [cite_start]David Navas Landa[cite: 14].

## üöÄ Key Features
* **Procedural Level Generation:** The track and obstacles are generated infinitely, ensuring that every run is different.
* **Intuitive Keyboard Controls:** Simple controls for fast-paced lateral movement and precise jumps.
* **High Score System:** The score increases with the distance traveled, motivating the player to beat their own record.
* **Dynamic Third-Person Camera:** A camera that follows the player to keep the action in view at all times.

## ‚öôÔ∏è Getting Started & Installation
To get a local copy up and running, follow these simple steps.

1.  Clone the repository to your local machine:
    ```bash
    git clone [YOUR-REPOSITORY-URL]
    ```
2.  Open **Unity Hub**.
3.  Click on the "Open" button and select the cloned project folder.
4.  Ensure you have a compatible version of the Unity Editor installed (2022.3.x or higher).
5.  Once the project is open, find the main scene inside the `Assets/Scenes/` folder.
6.  Press the **Play** button in the editor to start the game.
